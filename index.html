Definição.

O HTML*, sigla para (HyperText Markup Language), em português Linguagem de Marcação de HiperTexto, é uma Linguagem fundamental para estruturar páginas Web. Utilizada como base para criação de Websites, o HTML permite a organização de elementos fundamentais para estruturação do conteúdo oferecido pelo site. Por meio de tags, atributos, o HTML é capaz de definir a hierarquia e formatação do conteúdo apresentado, melhorando a experiência do usuário.

* Embora o HTML seja confundido com uma linguagem de programação, é importante compreender que ele não pode ser associado a esse tipo, pois uma linguagem de programação permite a criação de sites dinâmicos, por meio da utilização de um conjunto de comandos e sintaxe. Enquanto a linguagem de marcação nos permite somente a organização e estruturação do conteúdo.

História.

O HTML, Linguagem de Marcação de HiperTexto surgiu durante a década de 90, entre 1990 e 1995, criada por Tim Berners-Lee. A princípio se tratava de um projeto para compartilhamento de artigos científicos, Tim e seus colegas queriam uma maneira mais fácil de compartilhar esses documentos. Algo parecido com um repositório remoto, onde estaria armazenado artigos e documentos científicos, a partir disso surge a necessidade de um mecanismo para que essa comunicação se estabeleça.

Com isso temos a criação do HTML, pelo Tim e seu colega do CERN, Robert Cailliau, quando em 1989, conseguiram realizar a primeira comunicação entre o servidor, espaço onde os arquivos estão armazenados, e o cliente, que seria nosso navegador. Tornando um dos passos mais importantes para o então surgimento da internet. Desde então, o CERN passou a supervisionar e controlar a utilização da linguagem, até o ano de 1995, quando a W3C passou a supervisionar e controlar os padrões da linguagem.

O HTML então recebe atualizações até os dias atuais, tendo como sua versão mais recente o HTML 5, desenvolvido para aperfeiçoar a experiência de uso tanto para usuários quanto para desenvolvedores.

Principais características do HTML 5.

  • Suporte de áudio e vídeo em alto nível.
  • Utilização de um banco de dados Web.
  • Pode ser aprimorado e aperfeiçoado de forma independente.
  • Suporte para novos tipos de controle de formulários.

Como o HTML funciona?

Um site pode conter vários elementos dependendo da informação apresentada, como categorias, departamentos, informações de contato, entre outros. Esses elementos podem ser acessados em HTML's presentes no próprio site em questão ou em sites externos. O papel do HTML nesse processo todo seria linkar esses sites por meio de hiperlinks, além de organizar todo conteúdo apresentado em cada HTML, por meio das tags. Dessa forma, podemos fazer todo planejamento de um site por meio das tags existentes no HTML, temos tags de todos tipos, todas referentes à apresentação de conteúdos diversos, podendo ser textos, títulos, parágrafos, tabelas, links para acesso a outros HTML's, e muitos outros. Essas tags são constantemente atualizados para melhorar a experiência dos usuários e desenvolvedores.

Tags.

As tags assumem um papel fundamental na contrução de sites utilizando como base o HTML, com as tags podemos estruturar todos elementos de um site. As tags basicamente possuem duas formas principais, nós temos as tags de fechamento padrão, como por exemplo "<simbolo>conteúdo</>", podemos percerber que esse tipo de tag possui uma tag referente a abertura "<>", e uma tag referente ao fechamento "</>". Ou podemos ter as tags de "fechamento automático", elas basicamente não possuem uma tag de abertura e de fechamento, como a tag citada anteriormente, e sim uma tag única, como por exemplo a tag img, que pode ser utilizada da seguinte forma "<simbolo/>".

O HTML possui diversas tags, podendo ser utilizadas de várias formas. Temos tags textuais, para definir títulos, subtítulos, parágrafos, comentários, entre outros. Assim como tags para exibição de imagens, e conexão de outras páginas web. E algumas dessas tags podem receber atributos específicos, que podem modificar tamanho, cor, fonte, e diversas outras características. Esses atributos são adicionados diretamente na tag, como por exemplo a definição do tamanho de uma imagem "<img width="720"/>".

Os atributos possuem como formatação um padrão contendo dois elementos principais, o primeiro seria sua própria definição original, o que irá definir o que será modificado no texto. O segundo elemento seria o valor que passamos para este atributo, o valor que será responsável pela modifição em questão, como demonstrado no exemplo acima sobre as imagens. Podemos definir os tamanhos de cada imagem de forma bem específica, ou outras características dependendo do atributo utilizado.

Classes

As classes no HTML são destinadas a modificar e conectar nossas tags. Com elas podemos definir um tipo de indentificação para a tag, frequentemente utilizada para estilização, pois conseguimos modificar tags específicas, sem que haja modificação em outras partes do código. podemos definir uma classe da seguinte forma "<h1 id="Titulo_principal">conteúdo</>", assim conseguimos atribuir uma identificação específica ao título, referenciada com Titulo_principal, a partir disso conseguimos realizar alterações específicas na tag indicada.
